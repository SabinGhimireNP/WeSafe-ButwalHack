# WeSafe - Butwal Hack

WeSafe is a safety and emergency reporting platform built for Butwal Hackathon. It lets users quickly report incidents, receive alerts, and view emergency info on a map.

## Features

- Quick incident reporting
- Real-time alerts and notifications
- Emergency contact info
- Interactive map of reported incidents

## Tech Stack

- Frontend: ReactJS
- Backend: Node.js/Express
- Database: MongoDB
- Maps: Google Maps API

## Getting Started

1. Clone:
   ```bash
   git clone https://github.com/SabinGhimireNP/WeSafe-ButwalHack.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start:
   ```bash
   npm start
   ```
